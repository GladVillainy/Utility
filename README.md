# Java Utility Classes

A small set of simple utility classes used to make basic Java programming tasks easier.  
These tools are beginner-friendly and reused across small school projects.

## Included Utilities

### TextUI
Tools for console input:
- Show menus
- Read numbers, text, and yes/no answers
- Handles invalid user input safely

### FileIO
Basic file handling:
- Save a list of strings to a file
- Read lines from a file
- Uses try-with-resources to avoid resource leaks

### ApiUtility
Simple API helper:
- Send GET requests
- Read JSON as text
- Extract values from JSON responses

### DatabaseUtility
Static helper for simple database work:
- Connect to a database
- Read tables and columns
- Insert text or numbers

## Purpose
These utilities reduce repeated code and make small Java console projects easier to write and understand.
